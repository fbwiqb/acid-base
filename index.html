🎯 프로그램 소개
중화반응 시뮬레이션은 산과 염기의 중화 반응을 시각적으로 이해할 수 있는 교육용 프로그램입니다. 염산(HCl)과 수산화 나트륨(NaOH)의 중화 반응 과정을 이온 단위로 관찰하고, 온도 변화와 지시약 색 변화를 실시간으로 확인할 수 있습니다.
📚 기본 사용법
1단계: 용액 첨가
• 산 첨가 💧: 염산(HCl)을 추가합니다. H⁺와 Cl⁻ 이온이 생성됩니다
• 염기 첨가 💧: 수산화 나트륨(NaOH)을 추가합니다. Na⁺와 OH⁻ 이온이 생성됩니다
• 클릭할 때마다 2개의 이온이 추가됩니다

2단계: 중화 반응 관찰
• H⁺(빨간색)와 OH⁻(파란색) 이온이 만나면 물(H₂O) 분자가 생성됩니다
• 중화 반응이 일어나면 온도가 상승합니다 (발열 반응)
• 반응하지 않는 이온(Na⁺, Cl⁻)은 남아있습니다

3단계: 지시약으로 pH 확인
• BTB 용액: 산성(노란색) → 중성(초록색) → 염기성(파란색)
• 페놀프탈레인: 산성/중성(무색) → 염기성(분홍색)
• 없음: 온도 변화 그래프를 관찰할 수 있습니다

4단계: 그래프 분석
• 오른쪽 그래프에서 각 이온의 수 변화를 확인하세요
• 중화점에서는 H⁺와 OH⁻ 이온 수가 같아집니다
• 통계 패널에서 현재 이온 수와 생성된 물 분자 수를 확인할 수 있습니다
⚙️ 고급 기능
표시 옵션:
• 이온식: 화학 기호로 이온을 표시합니다 (H⁺, OH⁻, Na⁺, Cl⁻)
• 도형: 이온을 다양한 도형으로 표시합니다
  - H⁺: 빨간 삼각형 ▼
  - OH⁻: 파란 별 ★
  - Na⁺: 보라 사각형 ■
  - Cl⁻: 파란 원 ●
• 도형 뒤섞기 🔀: 도형 모드에서 각 이온에 할당된 도형을 랜덤하게 섞습니다
  - 이 기능으로 학생들이 색과 도형으로 이온을 구별하는 연습을 할 수 있습니다
  - 그래프 범례도 자동으로 변경됩니다

제어 옵션:
• 반응하지 않는 이온 숨기기: Na⁺와 Cl⁻ 이온을 숨겨 중화 반응에 집중할 수 있습니다
• 물분자 사라지지 않기: 생성된 물 분자를 계속 화면에 표시합니다
• 일시정지: 이온의 움직임을 멈춰 관찰할 수 있습니다
• 초기화: 모든 이온과 설정을 초기 상태로 되돌립니다
• 그래프 초기화: 그래프 데이터만 초기화합니다
💡 학습 팁
중화점 찾기:
• 산과 염기를 번갈아 가며 조금씩 첨가하세요
• 지시약 색이 중성을 나타낼 때가 중화점입니다
• 그래프에서 H⁺와 OH⁻ 이온 수가 같아지는 지점을 찾아보세요

실험 아이디어:
• 산만 계속 넣으면 어떻게 될까요?
• 중화 반응으로 온도가 얼마나 올라갈까요?
• 염의 종류가 다르면 중화 반응도 다를까요?
• 중화점 전후로 지시약 색 변화를 관찰해보세요
• 도형 뒤섞기 기능으로 색과 도형만으로 이온을 구별해보세요

주의사항:
• 이온을 너무 많이 추가하면 화면이 복잡해질 수 있습니다
• 필요시 초기화 버튼을 눌러 처음부터 시작하세요
• 실제 실험실에서는 보안경과 보호장구를 착용해야 합니다
🔬 과학적 배경
중화 반응이란?
산의 H⁺ 이온과 염기의 OH⁻ 이온이 결합하여 물(H₂O)을 생성하는 반응입니다.

화학 반응식:
HCl + NaOH → NaCl + H₂O
(이온 반응식: H⁺ + OH⁻ → H₂O)

중화열:
중화 반응은 발열 반응으로, 반응이 일어나면 열이 발생하여 온도가 상승합니다. 이 시뮬레이션에서는 중화 반응이 일어날 때마다 온도가 올라가는 것을 관찰할 수 있습니다.

구경꾼 이온:
Na⁺와 Cl⁻는 반응에 직접 참여하지 않고 구경만 하는 이온입니다. 이들은 중화 후 염(NaCl)을 형성합니다.
